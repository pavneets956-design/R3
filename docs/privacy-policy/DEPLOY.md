# Deploy Privacy Policy to GitHub Pages

1. Push this repo to GitHub (github.com/YOUR_USERNAME/r3-extension or similar)
2. Go to repo Settings → Pages
3. Source: Deploy from branch → main → /docs/privacy-policy
4. Click Save
5. Your privacy policy will be live at: https://YOUR_USERNAME.github.io/r3-extension/privacy-policy/

## Alternative: Vercel
1. Connect your GitHub repo to Vercel
2. Set root directory to docs/privacy-policy
3. Deploy

## After deploying
Update manifest.json privacy_policy_url field with the live URL.
